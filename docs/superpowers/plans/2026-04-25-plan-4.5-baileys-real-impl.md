# Plano 4.5 — BaileysClient real + ace command list-groups

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans + superpowers:test-driven-development.

**Goal:** Implementação real do `WhatsAppClient` via Baileys, mais o ace command que ajuda o admin a descobrir o `WHATSAPP_GROUP_JID` na configuração inicial.

1. **`BaileysClient`** — multi-file auth, `useMultiFileAuthState`, QR no terminal via `qrcode-terminal`, tracking de `connection.update`, reconnect com backoff (não reconecta em `loggedOut`).
2. **`start/whatsapp.ts`** preload (environment `web`) — chama `client.connect()` no boot e `disconnect()` em `app.terminating`.
3. **Ace command** `whatsapp:list-groups` — pareia, lista todos os grupos do WhatsApp (JID + nome), imprime no terminal.
4. **`.gitignore`** — `apps/api/storage/`.
5. **Atualizar `WhatsAppProvider`** — modo `real` agora retorna `BaileysClient` em vez de lançar.

**Architecture:**
- `BaileysClient` é stateful, deve ser singleton (provider já registra como singleton).
- Connect carrega creds; se vazias, Baileys emite QR via `connection.update.qr`.
- State machine: `'connecting' | 'open' | 'close'`. `isConnected()` retorna `state === 'open'`.
- Reconnect: se close por `loggedOut`, não reconecta — admin precisa apagar a sessão e parear de novo. Caso contrário, backoff 2s → 5s → 10s → cap 30s.
- **Sem testes automatizados** — Baileys depende de socket externo. Validação manual via `node ace whatsapp:list-groups` e observação do flow em dev.

**Pré-requisito:** Planos 4.1, 4.2, 4.3, 4.4 concluídos.

## Task 2: `BaileysClient`

**Files:**
- Create: `apps/api/app/integrations/whatsapp/baileys_client.ts`

- [ ] **Step 1: Implementação**

```ts
import { mkdir } from 'node:fs/promises'
import { existsSync } from 'node:fs'
import qrcode from 'qrcode-terminal'
import logger from '@adonisjs/core/services/logger'
import {
  default as makeWASocket,
  DisconnectReason,
  useMultiFileAuthState,
  type WASocket,
} from '@whiskeysockets/baileys'
import env from '#start/env'
import WhatsAppClient, { type WhatsAppMode } from './whatsapp_client.js'

type ConnState = 'connecting' | 'open' | 'close'

export default class BaileysClient extends WhatsAppClient {
  readonly mode: WhatsAppMode = 'real'
  private socket: WASocket | null = null
  private state: ConnState = 'close'
  private reconnectAttempt = 0
  private shuttingDown = false

  async connect(): Promise<void> {
    const jid = env.get('WHATSAPP_GROUP_JID')
    if (!jid) {
      throw new Error('WHATSAPP_GROUP_JID é obrigatório quando WHATSAPP_MODE=real')
    }

    const authPath = env.get('WHATSAPP_AUTH_PATH')
    if (!existsSync(authPath)) {
      await mkdir(authPath, { recursive: true })
    }

    await this.openSocket()
  }

  async disconnect(): Promise<void> {
    this.shuttingDown = true
    if (this.socket) {
      try {
        this.socket.end(undefined)
      } catch (err) {
        logger.warn({ err }, 'BaileysClient: erro ao encerrar socket')
      }
    }
    this.state = 'close'
  }

  isConnected(): boolean {
    return this.state === 'open'
  }

  async sendToGroup(text: string): Promise<void> {
    const jid = env.get('WHATSAPP_GROUP_JID')!
    if (!this.socket || this.state !== 'open') {
      throw new Error('BaileysClient: socket não conectado')
    }
    await this.socket.sendMessage(jid, { text })
  }

  /** Lista os grupos pareados — usado pelo ace command. */
  async fetchGroups(): Promise<{ jid: string; name: string }[]> {
    if (!this.socket || this.state !== 'open') {
      throw new Error('BaileysClient: socket não conectado')
    }
    const groups = await this.socket.groupFetchAllParticipating()
    return Object.values(groups).map((g) => ({ jid: g.id, name: g.subject ?? '(sem nome)' }))
  }

  private async openSocket(): Promise<void> {
    const authPath = env.get('WHATSAPP_AUTH_PATH')
    const { state, saveCreds } = await useMultiFileAuthState(authPath)

    this.state = 'connecting'
    this.socket = makeWASocket({
      auth: state,
      printQRInTerminal: false,
      // Baileys precisa de um logger pino-like
      logger: { level: 'silent' } as any,
    })

    this.socket.ev.on('creds.update', saveCreds)

    this.socket.ev.on('connection.update', (update) => {
      const { connection, lastDisconnect, qr } = update

      if (qr) {
        logger.info('BaileysClient: QR pendente — escaneie com o app do WhatsApp')
        qrcode.generate(qr, { small: true })
      }

      if (connection === 'open') {
        this.state = 'open'
        this.reconnectAttempt = 0
        logger.info('BaileysClient: conexão aberta')
      }

      if (connection === 'close') {
        this.state = 'close'
        const code = (lastDisconnect?.error as any)?.output?.statusCode
        const isLoggedOut = code === DisconnectReason.loggedOut

        if (this.shuttingDown) {
          logger.info('BaileysClient: encerrado durante shutdown')
          return
        }

        if (isLoggedOut) {
          logger.error(
            'BaileysClient: sessão deslogada — apague a pasta de auth e pareie de novo'
          )
          return
        }

        const delay = this.computeBackoff()
        logger.warn({ delayMs: delay, code }, 'BaileysClient: conexão caiu, reconectando')
        setTimeout(() => {
          this.openSocket().catch((err) => {
            logger.error({ err }, 'BaileysClient: falha ao reconectar')
          })
        }, delay)
      }
    })
  }

  private computeBackoff(): number {
    this.reconnectAttempt++
    const base = [2000, 5000, 10000, 20000, 30000]
    return base[Math.min(this.reconnectAttempt - 1, base.length - 1)]
  }
}
```

Notas:
- `printQRInTerminal: false` + `qrcode.generate(qr, { small: true })` no handler — controle explícito.
- `logger: { level: 'silent' }` evita ruído do Baileys (pino interno).
- `groupFetchAllParticipating()` retorna `Record<string, GroupMetadata>`; convertemos pra array.

```ts
import logger from '@adonisjs/core/services/logger'
import env from '#start/env'
import WhatsAppClient from '#integrations/whatsapp/whatsapp_client'

/**
 * Boot do client WhatsApp. Preload com environment=['web'],
 * pra não conectar em test/console/repl.
 *
 * Em modo `real`, conecta no boot. Em `stub`/`disabled`, no-op
 * (o client.connect() é não-bloqueante e barato pra esses modos).
 */

const mode = env.get('WHATSAPP_MODE')
const client = await app.container.make(WhatsAppClient)

try {
  await client.connect()
  logger.info({ mode }, 'whatsapp: client connected')
} catch (err) {
  logger.error({ err, mode }, 'whatsapp: connect failed')
  // Não derruba o app — cron jobs vão respeitar o gate (isConnected=false).
}

app.terminating(async () => {
  try {
    await client.disconnect()
  } catch (err) {
    logger.warn({ err }, 'whatsapp: disconnect error during shutdown')
  }
})
```

## Task 5: Ace command `whatsapp:list-groups`

**Files:**
- Create: `apps/api/commands/whatsapp/list_groups.ts`

- [ ] **Step 1: Comando**

```ts
import { BaseCommand } from '@adonisjs/core/ace'
import type { CommandOptions } from '@adonisjs/core/types/ace'
import env from '#start/env'
import WhatsAppClient from '#integrations/whatsapp/whatsapp_client'

export default class WhatsAppListGroups extends BaseCommand {
  static commandName = 'whatsapp:list-groups'
  static description = 'Lista os grupos do WhatsApp pareados (JID + nome) para configurar WHATSAPP_GROUP_JID'

  static options: CommandOptions = {
    startApp: true,
  }

  async run() {
    const mode = env.get('WHATSAPP_MODE')
    if (mode !== 'real') {
      this.logger.warning(`comando só funciona com WHATSAPP_MODE=real (atual: ${mode})`)
      return
    }

    const client = await this.app.container.make(WhatsAppClient)

    // Aguarda até 30s pela conexão (preload já tentou conectar; QR pode estar pendente)
    const start = Date.now()
    while (!client.isConnected()) {
      if (Date.now() - start > 30_000) {
        this.logger.error('timeout esperando conexão Baileys (30s)')
        return
      }
      await new Promise((r) => setTimeout(r, 500))
    }

    // BaileysClient tem fetchGroups; outros impls não. Cast por causa da port.
    const fetchGroups = (client as any).fetchGroups as (() => Promise<{ jid: string; name: string }[]>) | undefined
    if (!fetchGroups) {
      this.logger.error('client atual não suporta listagem de grupos')
      return
    }

    const groups = await fetchGroups.call(client)
    if (groups.length === 0) {
      this.logger.info('nenhum grupo encontrado — confirme o pareamento')
      return
    }

    this.logger.info(`${groups.length} grupos:`)
    for (const g of groups) {
      console.log(`${g.jid}\t${g.name}`)
    }
  }
}
```

Cast `as any` para `fetchGroups`: o método é específico do `BaileysClient` e não faz parte da port pública (`WhatsAppClient`). Alternativa: declarar opcional na port, mas isso vaza detalhe de impl. Ficando com cast aqui.

- Conexão **não-bloqueante**: connect falhar no boot (sessão expirada, rede instável) só deixa `isConnected=false`. Os jobs respeitam o gate; admin é avisado por `GET /whatsapp/status`.
- Sem testes automatizados pro `BaileysClient`: socket externo é frágil de mockar, ROI baixo. Validação manual cobre o que importa.
- `fetchGroups` específico do `BaileysClient` (não na port `WhatsAppClient`): vaza menos detalhe de impl. Cast `as any` no ace command.
