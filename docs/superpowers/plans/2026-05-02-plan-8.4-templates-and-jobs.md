# Plan 8.4 — Templates + Notifier + Jobs

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Anunciar "rodada dobrada" no grupo do WhatsApp em três momentos — abertura (`round_opened` + `round_opened_dm`), T-30min (`match_reminder`) e resultado pós-jogo (`match_finished`). Templates ganham campo opcional `pointsMultiplier`, jobs lêem do `match` e passam adiante.

**Architecture:** Cada template ganha `pointsMultiplier?: number` no `Input`. Quando `> 1`, render um header (templates pre-jogo) ou disclaimer (match_finished). Os 4 templates puros são unit-testados. `WhatsAppNotifier.notifyRoundOpenedToUser` ganha `pointsMultiplier?` no shape de args. Jobs (`OpenRoundJob`, `MatchReminderJob`, `SyncScoresJob`) leem `match.pointsMultiplier` e passam.

**Tech Stack:** Adonis 7, Japa unit + functional, TypeScript ESM.

**Spec:** `docs/superpowers/specs/2026-05-02-double-points-round-design.md`

**Pré-requisitos:** Plans 8.1, 8.2, 8.3 mergeados (schema + scoring + sync + finalizer + presenter prontos).

import type { DateTime } from 'luxon'

export interface RoundOpenedInput {
  roundNumber: number
  homeTeam: string
  awayTeam: string
  kickoffAt: DateTime
  pointsMultiplier?: number
}

const DOUBLE_POINTS_HEADER = '🔥 *RODADA VALENDO EM DOBRO!* (1º × 2º na tabela)'

export function roundOpenedMessage(input: RoundOpenedInput): string {
  const kickoff = input.kickoffAt.setZone('America/Sao_Paulo').toFormat("dd/MM 'às' HH:mm")
  const lines = [
    `📢 Rodada ${input.roundNumber} aberta! 🔥`,
    `⚽ Jogo: ${input.homeTeam} x ${input.awayTeam}`,
    `⏰ Início: ${kickoff}`,
  ]
  if ((input.pointsMultiplier ?? 1) > 1) {
    lines.unshift(DOUBLE_POINTS_HEADER)
  }
  return lines.join('\n')
}
```

import type { RankingEntry } from '#services/ranking_service'
import type { RoundScoreEntry } from './types.js'

export interface MatchFinishedInput {
  roundNumber: number
  homeTeam: string
  awayTeam: string
  finalHome: number
  finalAway: number
  roundScores: RoundScoreEntry[]
  seasonRanking: RankingEntry[]
  pointsMultiplier?: number
}

function ptsLabel(n: number): string {
  return n === 1 ? '1 pt' : `${n} pts`
}

function exactScoresLabel(n: number): string {
  return n === 1 ? '1 placar exato' : `${n} placares exatos`
}

export function matchFinishedMessage(input: MatchFinishedInput): string {
  const final = `🏁 Final: ${input.homeTeam} ${input.finalHome} x ${input.finalAway} ${input.awayTeam}`
  const disclaimer =
    (input.pointsMultiplier ?? 1) > 1
      ? 'ℹ️ Foi rodada dobrada — pontuação multiplicada por 2.'
      : null

  const roundLines = input.roundScores.map((e) => `${e.name} ${e.emoji} — ${ptsLabel(e.points)}`)
  const roundBlock = `Pontuação da rodada ${input.roundNumber}:\n${roundLines.join('\n')}`

  const rankingLines = input.seasonRanking.map(
    (e, idx) =>
      `${idx + 1}. ${e.name} ${e.emoji} — ${ptsLabel(e.totalPoints)} (${exactScoresLabel(e.exactScoresCount)})`
  )
  const rankingBlock = `🏆 Ranking da temporada:\n${rankingLines.join('\n')}`

  const blocks = disclaimer
    ? [final, disclaimer, roundBlock, rankingBlock]
    : [final, roundBlock, rankingBlock]
  return blocks.join('\n\n')
}
```
