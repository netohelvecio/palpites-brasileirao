# Plan 9.1 — Schema + Shared Types (Tied Featured Match Poll)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adicionar o status `awaiting_pick` ao enum `round_status`, criar a tabela `round_match_candidates`, propagar `RoundStatus.AWAITING_PICK` e `MatchCandidateView` em `@palpites/shared`, e criar o model + factory `RoundMatchCandidate`.

**Architecture:** Duas migrations (uma só pra `ALTER TYPE` fora de transaction; outra pra criar tabela). Update do shared package (status + view type). Model novo segue padrão Lucid local (`extends RoundMatchCandidateSchema` + `@beforeCreate` UUID + soft-delete scope). Factory pra setup de testes futuros.

**Tech Stack:** Adonis 7, Lucid migrations, Postgres native enum, TypeScript ESM, Japa.

**Spec:** `docs/superpowers/specs/2026-05-07-tied-featured-match-poll-design.md`

import { BaseSchema } from '@adonisjs/lucid/schema'

export default class extends BaseSchema {
  // ALTER TYPE ... ADD VALUE precisa rodar fora de transaction no Postgres.
  static disableTransactions = true

  async up() {
    this.schema.raw(`ALTER TYPE round_status ADD VALUE IF NOT EXISTS 'awaiting_pick'`)

  async down() {

    // Down é no-op intencional. Para reverter de verdade, criar migration nova

  }
}

### Task 4: Model `RoundMatchCandidate`

**Files:**
- Create: `apps/api/app/models/round_match_candidate.ts`

- [ ] **Step 1: Criar o model**

```ts
import { randomUUID } from 'node:crypto'
import { beforeCreate, beforeFetch, beforeFind } from '@adonisjs/lucid/orm'
import type { ModelQueryBuilderContract } from '@adonisjs/lucid/types/model'
import { RoundMatchCandidateSchema } from '#database/schema'

export default class RoundMatchCandidate extends RoundMatchCandidateSchema {
  static selfAssignPrimaryKey = true

  @beforeCreate()
  static assignUuid(row: RoundMatchCandidate) {
    if (!row.id) row.id = randomUUID()
  }

  @beforeFind()
  @beforeFetch()
  static softDeleteScope(query: ModelQueryBuilderContract<typeof RoundMatchCandidate>) {
    query.where('is_deleted', false)
  }
}
```
