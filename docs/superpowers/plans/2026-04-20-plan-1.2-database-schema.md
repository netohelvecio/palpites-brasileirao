# Plano 1.2 — Database Schema (Migrations)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development ou superpowers:executing-plans.

**Goal:** Criar todas as 6 migrations (users, seasons, rounds, matches, guesses, scores) com UUID (`gen_random_uuid()`) e soft delete (`is_deleted`).

**Architecture:** PostgreSQL com extensão `pgcrypto` pra UUID. Enums nativos pra `round_status` e `match_status`. FKs com `ON DELETE CASCADE`.

**Tech Stack:** Lucid migrations, Postgres 16.

**Spec:** `docs/superpowers/specs/2026-04-20-palpites-brasileirao-design.md` (seção "Modelo de dados").

**Pré-requisito:** Plano 1.1 concluído.

**Critério de aceitação:** `node ace migration:run` aplica as 6 migrations sem erro; `psql` mostra as tabelas com colunas corretas.

> ⚠️ **Commits manuais.**

Renomear pra iniciar com `0001_` se o Adonis não gerou com esse prefixo (o nome do arquivo começa com timestamp — pode renomear manualmente pra manter ordem explícita, ou deixar o timestamp).

- [ ] **Step 2: Escrever migration**

Arquivo: `apps/api/database/migrations/0001_create_users_table.ts`
```ts
import { BaseSchema } from '@adonisjs/lucid/schema'

export default class extends BaseSchema {
  protected tableName = 'users'

  async up() {
    this.schema.raw('CREATE EXTENSION IF NOT EXISTS "pgcrypto"')

    this.schema.createTable(this.tableName, (table) => {
      table.uuid('id').primary().defaultTo(this.raw('gen_random_uuid()'))
      table.string('name', 120).notNullable()
      table.string('whatsapp_number', 20).notNullable().unique()
      table.string('emoji', 10).notNullable()
      table.boolean('is_admin').notNullable().defaultTo(false)
      table.boolean('is_deleted').notNullable().defaultTo(false)
      table.timestamp('created_at', { useTz: true }).notNullable().defaultTo(this.now())
      table.timestamp('updated_at', { useTz: true }).notNullable().defaultTo(this.now())
    })
  }

  async down() {
    this.schema.dropTable(this.tableName)
  }
}
```

export default class extends BaseSchema {
  protected tableName = 'rounds'

  async up() {
    this.schema.createTable(this.tableName, (table) => {
      table.uuid('id').primary().defaultTo(this.raw('gen_random_uuid()'))
      table.uuid('season_id').notNullable().references('id').inTable('seasons').onDelete('CASCADE')
      table.integer('number').notNullable()
      table.enum('status', ['pending', 'open', 'closed', 'finished'], {

        enumName: 'round_status',

      table.timestamp('created_at', { useTz: true }).notNullable().defaultTo(this.now())
      table.timestamp('updated_at', { useTz: true }).notNullable().defaultTo(this.now())

      table.unique(['season_id', 'number'])
    })
  }

  async down() {
    this.schema.dropTable(this.tableName)
    this.schema.raw('DROP TYPE IF EXISTS round_status')
