# Plano 1.1 — Monorepo & Adonis Scaffold

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development ou superpowers:executing-plans.

**Goal:** Inicializar o monorepo (pnpm + Turborepo), subir Postgres via Docker, fazer scaffold do app Adonis em `apps/api` e criar o package `shared`.

**Architecture:** Monorepo com `apps/api` (Adonis 6) e `packages/shared` (tipos compartilhados). Postgres 16 roda em Docker com dois bancos (dev e test).

**Tech Stack:** pnpm workspaces, Turborepo, Adonis 6, Postgres 16, Docker Compose.

**Spec de referência:** `docs/superpowers/specs/2026-04-20-palpites-brasileirao-design.md`

**Critério de aceitação:**
- `docker compose up -d` sobe Postgres dev e test.
- `pnpm --filter @palpites/api dev` sobe API em `localhost:3333`.
- `pnpm --filter @palpites/shared lint` passa sem erro.

> ⚠️ **Commits manuais:** nenhuma task inclui `git commit`. Revise o diff e commite quando quiser.

## Task 1: Inicializar monorepo (pnpm + Turborepo)

**Files:**
- Create: `package.json`
- Create: `pnpm-workspace.yaml`
- Create: `turbo.json`
- Create: `.gitignore`
- Create: `docker-compose.yml`

- [ ] **Step 1: Inicializar git na raiz**

```bash
cd palpites-brasileirao
git init
```

- [ ] **Step 2: Criar `package.json` raiz**

Arquivo: `package.json`
```json
{
  "name": "palpites-brasileirao",
  "private": true,
  "version": "0.1.0",
  "scripts": {
    "dev": "turbo run dev",
    "build": "turbo run build",
    "test": "turbo run test",
    "lint": "turbo run lint"
  },
  "devDependencies": {
    "turbo": "^2.0.0",
    "typescript": "^5.5.0"
  },
  "packageManager": "pnpm@9.0.0",
  "engines": {
    "node": ">=20.11.0"
  }
}
```

- [ ] **Step 3: Criar `pnpm-workspace.yaml`**

Arquivo: `pnpm-workspace.yaml`
```yaml
packages:
  - 'apps/*'
  - 'packages/*'
```

- [ ] **Step 4: Criar `turbo.json`**

Arquivo: `turbo.json`
```json
{
  "$schema": "https://turbo.build/schema.json",
  "tasks": {
    "build": { "dependsOn": ["^build"], "outputs": ["build/**", "dist/**"] },
    "dev": { "cache": false, "persistent": true },
    "test": { "dependsOn": ["^build"] },
    "lint": {}
  }
}
```

- [ ] **Step 5: Criar `.gitignore`**

Arquivo: `.gitignore`
```
node_modules/
.turbo/
build/
dist/
.env
.env.local
*.log
.DS_Store
storage/whatsapp-auth/
apps/api/tmp/
coverage/
```

- [ ] **Step 6: Criar `docker-compose.yml`**

Arquivo: `docker-compose.yml`
```yaml
services:
  postgres:
    image: postgres:16-alpine
    container_name: palpites_postgres
    ports: ["5432:5432"]
    environment:
      POSTGRES_USER: palpites
      POSTGRES_PASSWORD: palpites
      POSTGRES_DB: palpites_dev
    volumes:
      - postgres_data:/var/lib/postgresql/data

  postgres_test:
    image: postgres:16-alpine
    container_name: palpites_postgres_test
    ports: ["5433:5432"]
    environment:
      POSTGRES_USER: palpites
      POSTGRES_PASSWORD: palpites
      POSTGRES_DB: palpites_test

volumes:
  postgres_data:
```

- [ ] **Step 7: Instalar deps e subir Postgres**

```bash
pnpm install
docker compose up -d
```

Verificar: `docker ps` mostra `palpites_postgres` e `palpites_postgres_test`.

## Task 2: Scaffold Adonis API

**Files:**
- Create: `apps/api/` via CLI
- Modify: `apps/api/package.json`
- Create: `apps/api/.env.example`

- [ ] **Step 1: Criar app Adonis**

```bash
cd apps
pnpm create adonisjs@latest api -- --kit=api --db=postgres --auth-guard=access_tokens
cd ..
```

Responder prompts: Postgres, testes com Japa.

- [ ] **Step 2: Ajustar nome no `apps/api/package.json`**

Mudar `"name"` para `"@palpites/api"`.

- [ ] **Step 3: Remover `.git` interno**

```bash
rm -rf apps/api/.git
```

- [ ] **Step 4: Criar `.env.example`**

Arquivo: `apps/api/.env.example`
```
TZ=America/Sao_Paulo
PORT=3333
HOST=localhost
LOG_LEVEL=info
APP_KEY=gerar-via-node-ace-generate-key
NODE_ENV=development

DB_HOST=localhost
DB_PORT=5432
DB_USER=palpites
DB_PASSWORD=palpites
DB_DATABASE=palpites_dev

ADMIN_API_TOKEN=dev-admin-token-change-me
```

- [ ] **Step 5: Copiar `.env` e gerar app key**

```bash
cp apps/api/.env.example apps/api/.env
cd apps/api && node ace generate:key
```

Colar a key no `.env`.

- [ ] **Step 6: Subir API em dev e verificar**

```bash
cd ../..
pnpm --filter @palpites/api dev
```

Verificar: `curl http://localhost:3333/` devolve resposta default Adonis. Parar com Ctrl+C.
