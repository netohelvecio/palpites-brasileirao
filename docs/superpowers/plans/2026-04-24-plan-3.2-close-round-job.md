# Plano 3.2 — CloseRoundJob

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans.

**Goal:** Implementar o job que roda a cada 5 min e flipa `round.status: open → closed` quando o kickoff do match já passou (`match.kickoff_at <= now`).

**Architecture:**
- `CloseRoundJob` em `app/jobs/close_round_job.ts` via DI.
- `RoundRepository.listOpenPastKickoff(date)` — joins round+match, filtra status=open e match.kickoff_at<=date, respeita soft-delete.
- Scheduler registra cron `*/5 * * * *` TZ BRT.

**Pré-requisito:** Plano 3.1 concluído (infra de scheduler existe).

**Critério de aceitação:**
- Job flipa rounds qualificados; deixa os demais intocados.
- Testes funcionais cobrem: happy path (open + kickoff passado → closed), kickoff futuro ignorado, rounds em outros status ignorados.

```ts
listOpenPastKickoff(date: DateTime) {

    .whereHas('match', (m) => {
```

## Task 2: `CloseRoundJob`

**Files:**
- Create: `apps/api/app/jobs/close_round_job.ts`

- [ ] **Step 1: Criar job**

```ts
import { inject } from '@adonisjs/core'
import { DateTime } from 'luxon'
import logger from '@adonisjs/core/services/logger'
import RoundRepository from '#repositories/round_repository'

export interface CloseRoundReport {
  closedCount: number
  closedRoundIds: string[]
}

@inject()
export default class CloseRoundJob {
  constructor(private roundRepository: RoundRepository) {}

  async run(now: DateTime = DateTime.now()): Promise<CloseRoundReport> {
    const rounds = await this.roundRepository.listOpenPastKickoff(now)
    const closedRoundIds: string[] = []

    for (const round of rounds) {
      await this.roundRepository.update(round, { status: 'closed' } as any)
      closedRoundIds.push(round.id)
    }

    const report = { closedCount: closedRoundIds.length, closedRoundIds }
    logger.info(report, 'CloseRoundJob: finished')
    return report
  }
}
```

```ts
import { test } from '@japa/runner'
import { DateTime } from 'luxon'
import app from '@adonisjs/core/services/app'
import testUtils from '@adonisjs/core/services/test_utils'
import CloseRoundJob from '#jobs/close_round_job'
import { SeasonFactory } from '#factories/season_factory'
import { RoundFactory } from '#factories/round_factory'
import { MatchFactory } from '#factories/match_factory'

test.group('CloseRoundJob', (group) => {
  group.each.setup(() => testUtils.db().wrapInGlobalTransaction())

  test('flipa round open → closed quando kickoff passou', async ({ assert }) => {
    const season = await SeasonFactory.create()
    const round = await RoundFactory.merge({
      seasonId: season.id,
      status: 'open',
    }).create()
    await MatchFactory.merge({
      roundId: round.id,
      kickoffAt: DateTime.now().minus({ minutes: 10 }),
    }).create()

    const job = await app.container.make(CloseRoundJob)
    const report = await job.run()

    assert.equal(report.closedCount, 1)
    const fresh = await round.refresh()
    assert.equal(fresh.status, 'closed')
  })

  test('ignora round open com kickoff futuro', async ({ assert }) => {
    const season = await SeasonFactory.create()
    const round = await RoundFactory.merge({
      seasonId: season.id,
      status: 'open',
    }).create()
    await MatchFactory.merge({
      roundId: round.id,
      kickoffAt: DateTime.now().plus({ hours: 3 }),
    }).create()

    const job = await app.container.make(CloseRoundJob)
    const report = await job.run()

    assert.equal(report.closedCount, 0)
    const fresh = await round.refresh()
    assert.equal(fresh.status, 'open')
  })

  test('ignora rounds pending / closed / finished mesmo com kickoff passado', async ({
    assert,
  }) => {
    const season = await SeasonFactory.create()
    const rounds = await Promise.all(
      (['pending', 'closed', 'finished'] as const).map(async (status, i) => {
        const r = await RoundFactory.merge({
          seasonId: season.id,
          number: i + 1,
          status,
        }).create()
        await MatchFactory.merge({
          roundId: r.id,
          kickoffAt: DateTime.now().minus({ hours: 1 }),
        }).create()
        return r
      })
    )

    const job = await app.container.make(CloseRoundJob)
    const report = await job.run()

    assert.equal(report.closedCount, 0)
    for (const r of rounds) {
      const fresh = await r.refresh()
      assert.notEqual(fresh.status, 'closed')
    }
  })
})
```

## Resumo do Plano 3.2

- Query elegante via `whereHas('match', ...)` em Lucid — reusa soft-delete scope do model Match.
- Job sem erro isolado por round (raro falhar; update simples).
- Scheduler de 5 min casa com `betting_policy` já defensivo (bloqueia palpites pós-kickoff mesmo se round atrasar a fechar).

**Decisões:**
- `now` recebe parâmetro no `run()` pra facilitar tests determinísticos (default = `DateTime.now()`).
- Sem tratamento per-round de erro: se a query falhar, o job inteiro loga e morre; próximo tick pega.
